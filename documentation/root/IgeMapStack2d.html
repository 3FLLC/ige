IgeMapStack2d CLASS

Declared in IgeMapStack2d.js

Description
Creates a new map that has two dimensions (x and y) to it's data and allows multiple items to be stored or "stacked" on a single x, y map position.
Extends
IgeClass

Methods

tileDataArray

Gets / sets the data stored at the specified map tile co-ordinates. If data already exists at the specified co-ordinates, it is replaced with the passed data.
  • {Number}x
  • {Number}y
  • {Array}val The array of data items to set at the specified co-ordinates.

tileDataAtIndexArray

Gets the data stored at the specified co-ordinates and index.
  • {Number}x
  • {Number}y
  • {Number}index

pushArray

Adds a data item to the specified map tile co-ordinates.
  • {Number}x
  • {Number}y
  • {*}val The data to add.

pullArray

Removes a data item from the specified map tile co-ordinates.
  • {Number}x
  • {Number}y
  • {*}val The data to remove.

collisionArray

Checks if the tile area passed has any data stored in it. If so, returns true, otherwise false.
  • {}x
  • {}y
  • {}width
  • {}height

clearDataArray

Clears any data set at the specified map tile co-ordinates.
  • {}x
  • {}y

mapDataArray

Gets / sets the map's tile data.
  • {Array}val The map data array.
