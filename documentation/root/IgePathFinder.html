IgePathFinder CLASS

Declared in {file}

Description
Creates a new path using the A* path-finding algorithm.
Extends
IgeClass

Methods

aStarArray

Uses the A* algorithm to generate path data between two points.
  • {IgeCollisionMap2d}tileMap The tile map to use when generating the path.
  • {IgePoint}startPoint The point on the map to start path-finding from.
  • {IgePoint}endPoint The point on the map to try to path-find to.
  • {Function}comparisonCallback The callback function that will decide if each tile that is being considered for use in the path is allowed or not based on the tile map's data stored for that tile which is passed to this method as the first parameter. Must return a boolean value.
  • {Boolean}allowSquare Whether to allow neighboring tiles along a square axis. Defaults to true if undefined.
  • {Boolean}allowDiagonal Whether to allow neighboring tiles along a diagonal axis. Defaults to false if undefined.

_getNeighboursArray

Get all the neighbors of a node for the A* algorithm.
  • {IgePathNode}currentNode The current node along the path to evaluate neighbors for.
  • {IgePathNode}endPoint The end point of the path.
  • {IgeCollisionMap2d}tileMap The tile map to use when evaluating neighbours.
  • {Function}comparisonCallback The callback function that will decide if the tile data at the neighbouring node is to be used or not. Must return a boolean value.
  • {Boolean}allowSquare Whether to allow neighboring tiles along a square axis.
  • {Boolean}allowDiagonal Whether to allow neighboring tiles along a diagonal axis.

_heuristicArray

The heuristic to add to a movement cost for the A* algorithm.
  • {Number}x1 The first x co-ordinate.
  • {Number}y1 The first y co-ordinate.
  • {Number}x2 The second x co-ordinate.
  • {Number}y2 The second y co-ordinate.
