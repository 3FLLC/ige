IgeTextureMap CLASS

Declared in {file}

Description
Texture maps provide a way to display textures across a tile map.
Extends
IgeTileMap2d

Methods

addTextureArray

Adds a texture to the texture map's internal texture list so that it can be referenced via an index so that the texture map's data will be something like [[textureId, textureCell]] or a real world example: [[0, 1], [1, 1]].

loadMapArray

Reads the map data from a standard map object and fills the map with the data found.

saveMap

Returns a map JSON string that can be saved to a data file and loaded with the loadMap() method.

tickArray

Handles rendering the texture map during engine tick events.
  • {}ctx

_newTileEntity

Creates an entity object that a texture can use to render itself. This is basically a dummy object that has the minimum amount of data in it that a texture requires to render such as geometry, texture cell and rendering position.
